Estrutura Básica</h1>

<p>O HTML é formado por tags, as tags que formam a estrutura básica são <code>&lt;html&gt;</code>, <code>&lt;head&gt;</code>, <code>&lt;body&gt;</code>, <code>&lt;title&gt;</code>,<code>&lt;script&gt;</code>, <code>&lt;meta&gt;</code> e <code>&lt;link&gt;</code>.</p>

<p>A tag <code>&lt;html&gt;</code> informa ao navegador que se trata de um código HTML.
A tag <code>&lt;head&gt;</code> é onde são inseridas informações que serão carregadas pelo navegador antes da página em si. Dentro de <code>&lt;head&gt;</code> é inserida a tag <code>&lt;title&gt;</code>, que determina o título da página, que será exibido na barra de título.
Outras tags que são inseridas dentro de <code>&lt;head&gt;</code> são:
<code>&lt;meta&gt;</code>: determina elementos de configuração, como codificação de letra e idioma da página.
<code>&lt;link&gt;</code>: usada para carregar folhas de estilo na página.
<code>&lt;script&gt;</code>: usada para carregar scripts na página.
Abaixo de <code>&lt;head&gt;</code> é inserida a tag <code>&lt;body&gt;</code>. Em <code>&lt;body&gt;</code> são inseridos os elementos que farão parte do corpo da página e serão exibidos ao usuário.</p>

<p><strong>HTML</strong>
Para a criação do menu, são utilizadas as tags <code>&lt;nav&gt;</code>, <code>&lt;div&gt;</code>, <code>&lt;a&gt;</code>, <code>&lt;button&gt;</code>, <code>&lt;ul&gt;</code> e <code>&lt;li&gt;</code>.

Além delas são usadas outras tags:
<code>&lt;a&gt;</code>: é usada para a criação de links.
<code>&lt;button&gt;</code>: usada para criação de botões.
<code>&lt;ul&gt;</code>: usada para criação de listas não ordenadas.
<code>&lt;li&gt;</code>: especifica cada item de uma lista.</p>

<p><strong>CSS</strong>
